# Escopo de uso

Este projeto é somente um player de mídia para listas próprias ou autorizadas pelo usuário.

Não há:

- Listas IPTV prontas.
- Canais ou URLs de conteúdo protegido.
- Bypass de DRM.
- Bypass de bloqueios de loja.
- Ocultação, camuflagem ou mecanismos anti-análise.
- Assets, marca ou layout copiados de apps comerciais.

Ao publicar, mantenha termos de uso claros e remova qualquer lista de origem duvidosa adicionada por terceiros.
